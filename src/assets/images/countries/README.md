# Country Images Directory

Place country-specific images in this directory with the following naming convention:

- `uk.jpg` - United Kingdom image
- `spain.jpg` - Spain image
- `usa.jpg` - United States image
- `uae.jpg` - United Arab Emirates image
- `pakistan.jpg` - Pakistan image

The images will be automatically loaded on the respective country pages. If an image is not found, the page will gracefully fall back to a gradient background.

Supported formats: `.jpg`, `.png`, `.webp`
